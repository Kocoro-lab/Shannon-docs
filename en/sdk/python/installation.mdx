## Installation

### Via pip (Recommended)

```bash
pip install shannon-sdk
```

### From Source

```bash
git clone https://github.com/Kocoro-lab/Shannon.git
cd Shannon/clients/python
pip install -e .
```

### Requirements

- Python 3.9 or higher
- pip or poetry for package management

## Verify Installation

```python
import shannon
print(shannon.__version__)
```

## Basic Setup

### Import the Client

```python
from shannon import ShannonClient

# HTTP-only client (defaults to Gateway at http://localhost:8080)
client = ShannonClient()
```

### With Authentication

```python
from shannon import ShannonClient

client = ShannonClient(api_key="sk_test_123456")
```

### Using Environment Variables

The SDK does not auto-read environment variables. Use your own config loader and pass values explicitly:

```python
import os
from shannon import ShannonClient

client = ShannonClient(
    base_url=os.getenv("SHANNON_BASE_URL", "http://localhost:8080"),
    api_key=os.getenv("SHANNON_API_KEY"),
)
```

## Quick Example

```python
from shannon import ShannonClient

client = ShannonClient()

handle = client.submit_task(query="What is the capital of France?")
final = client.wait(handle.task_id, timeout=60)
print("Result:", final.result)
```

## Configuration Options

### Client Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `base_url` | str | `http://localhost:8080` | Gateway base URL (HTTP only) |
| `api_key` | str | None | API key sent as `X-API-Key` |
| `bearer_token` | str | None | Alternative auth header `Authorization: Bearer` |
| `default_timeout` | float | 30.0 | Default HTTP timeout (seconds) |

### Full Configuration

```python
client = ShannonClient(
    base_url="http://localhost:8080",
    api_key="sk_live_123456",
    default_timeout=60.0,
)
```

## Async Client

For async applications, use `AsyncShannonClient`:

```python
from shannon import AsyncShannonClient
import asyncio

async def main():
    async with AsyncShannonClient(base_url="http://localhost:8080") as client:
        handle = await client.submit_task(query="Analyze data")
        status = await client.wait(handle.task_id)
        print(status.result)

asyncio.run(main())
```

## Environment Variables

The SDK does not auto-load env vars. These are common variables you can define and pass to the client:

| Variable | Description | Example |
|----------|-------------|---------|
| `SHANNON_BASE_URL` | Gateway HTTP base URL | `http://localhost:8080` |
| `SHANNON_API_KEY` | API key | `sk_test_123456` |
| `SHANNON_TIMEOUT` | Default timeout (seconds) | `60` |

### .env Example

```bash
SHANNON_BASE_URL=http://localhost:8080
SHANNON_API_KEY=sk_test_123456
SHANNON_TIMEOUT=60
```

Load with python-dotenv:

```python
from dotenv import load_dotenv
import os
from shannon import ShannonClient

load_dotenv()

client = ShannonClient(
    base_url=os.getenv("SHANNON_BASE_URL", "http://localhost:8080"),
    api_key=os.getenv("SHANNON_API_KEY"),
    default_timeout=float(os.getenv("SHANNON_TIMEOUT", "60")),
)
```

## Logging

Enable SDK logging:

```python
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger('shannon')
```

Debug mode:

```python
import logging

logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s'
)
```
